End-to-end analytics platform built on ~205,000 real traffic accidents in Serbia (2020–2026). Covers the full data science pipeline: EDA, feature engineering, ML modeling, REST API, and an interactive React dashboard.

- **ML:** XGBoost classifier with AUC-ROC of 0.80, custom threshold tuning for safety-critical recall
- **Backend:** FastAPI + SQLAlchemy + PostgreSQL
- **Frontend:** React 19, Recharts, Leaflet maps
- **Infra:** Dockerized, deployment-ready
